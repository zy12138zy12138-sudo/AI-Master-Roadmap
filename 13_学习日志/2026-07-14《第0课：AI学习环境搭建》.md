# 四、Git、GitHub、Obsidian关系


```
Obsidian
（学习笔记）

      ↓

Markdown文件

      ↓

Git
（版本管理）

      ↓

GitHub
（远程保存）


VS Code
（代码开发）
```


核心理解：

- Git负责记录文件变化
- GitHub负责远程保存
- Obsidian负责知识整理
- VS Code负责代码开发


---

# 五、Git常用命令


|命令|作用|
|-|-|
|git --version|查看Git版本|
|git init|初始化Git仓库|
|git status|查看文件状态|
|git remote -v|查看GitHub连接|
|git add .|添加修改|
|git commit -m|提交版本|
|git push|上传GitHub|
|git branch|查看分支|


常用流程：

```
修改文件

↓

git add .

↓

git commit

↓

git push
```

# 六、遇到问题


## Git连接问题

问题：

GitHub push失败。


原因：

网络连接异常。


解决：

重新执行push成功。


---

## Obsidian配置文件问题

问题：

Git检测：

```
.obsidian/workspace.json
```


原因：

该文件记录Obsidian界面状态。


处理：

后续通过 `.gitignore` 管理，不上传配置文件。
